# Kenny Portes 

## Overview 
This is my website of a small business for selling German automobiles (Audi, BMW, Mercedes, Porsche, Opel,
and Volkswagen). Each vehicle brand will have a link to a separate page that will have a list of a few 
vehicles within that brand. The link will diaplay the price, specification, and images of each vehicle.
Along with a form for user to enter there information, and a link to navigate back to the home page.
